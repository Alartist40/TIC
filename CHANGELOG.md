# Changelog

All notable changes to the Tokyo International Seventh-day Adventist Church website.

## [4.0.0] - 2026-01-26

### Design Pivot
- **TIC 11 Adoption**: Complete migration to the "TIC 11" design language as the primary UI.
- **Structure**: Replaced the previous layout with the TIC 11 HTML/CSS structure while retaining the robust Google Sheets integration logic.

### Features
- **Integrated Data**: "Read More" buttons now trigger modals populated with live data from Google Sheets (About, Ministries, Schedule).
- **Responsive 7th Grid**: Verified 6:1 column ratio on mobile and desktop using TIC 11's flex/grid hybrid approach.

## [3.0.0] - 2026-01-26
- Vanilla JS Rebuild (TIC 12 Base).

## [2.0.0] - 2026-01-23
