# Tokyo International Seventh-day Adventist Church Website

Welcome to the Tokyo International Seventh-day Adventist Church website project! This project aims to create an informative and engaging online presence for our church community.

## Project Structure

The project is organized as follows:

```
tokyo-intl-sda-church-website
├── index.html               # Main entry point for the website
├── sections                 # Contains various sections of the website
│   ├── hero.html           # Welcome hero section
│   ├── about.html          # About the church
│   ├── beliefs.html        # Core beliefs of the church
│   ├── pastor.html         # Information about the church pastor
│   └── ministry.html       # Ministry cards and Google Calendar
├── assets                   # Contains assets for the website
│   ├── css
│   │   └── styles.css      # CSS styles for the website
│   └── js
│       └── main.js        # JavaScript functionality for the website
└── README.md               # Project documentation
```

## Features

- **Welcome Hero Section**: A visually appealing introduction to the church.
- **About Section**: Detailed information about the church's history, mission, and community involvement.
- **Core Beliefs**: An outline of the church's fundamental beliefs and doctrines.
- **Pastor Information**: Background and contributions of Pastor Guenji Imayuki and Eliane Hosokawa Imayuki.
- **Ministry Cards**: Highlights of various church ministries and activities.
- **Upcoming Events**: An embedded Google Calendar to keep the community informed about upcoming events.

## Setup Instructions

1. Clone the repository to your local machine.
2. Open the `index.html` file in a web browser to view the website.
3. Customize the content in the `sections` directory as needed.
4. Modify the styles in `assets/css/styles.css` to fit your design preferences.
5. Implement any JavaScript functionality in `assets/js/main.js`.
